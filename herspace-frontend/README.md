# HerSpace 🌸

**HerSpace** is a personalized PCOD/PMOS Risk-Awareness & Wellness Guidance Platform designed to help users understand early body signals, track wellness patterns, and take timely action through a supportive digital experience.

> Track the signals. Understand the risk. Act early.

## Overview

HerSpace is not designed to diagnose PCOD/PMOS.  
It focuses on early awareness, personalized wellness guidance, and preventive action by converting symptoms, lifestyle inputs, cycle patterns, and skin-related signals into structured insights.

The platform provides a calm, user-friendly dashboard where users can complete risk screening, track periods, explore wellness zones, view AI-generated insights, and receive doctor-guided support when needed.

## Key Features

- Personalized PCOD/PMOS risk screening
- Symptom and lifestyle-based assessment
- Risk score and confidence score
- Personalized wellness zones
- AI-generated wellness insights
- Diet, movement, and self-care action plan
- Period tracking and cycle journey support
- AI skin health analyzer concept
- Mini check-ins and full re-check workflow
- Doctor-guided support section
- Responsive and visually supportive dashboard

## Tech Stack

- React.js
- Vite
- JavaScript
- HTML5
- CSS3

## Project Purpose

This project was built as a health-tech learning project to explore how digital products can support women’s health awareness through responsible design, personalized guidance, and preventive healthcare thinking.

The goal is to make users feel informed and supported, not overwhelmed or fearful.

## Future Scope

Going ahead, HerSpace can evolve from PCOD/PMOS risk awareness into a broader women’s wellness support system — one that helps users better understand long-term hormonal, lifestyle, and metabolic health patterns in a responsible and non-fear-based way.

## Medical Disclaimer

HerSpace is an awareness and wellness guidance platform. It does not provide medical diagnosis, treatment, or professional medical advice. Users should consult a qualified healthcare professional for medical concerns, diagnosis, or treatment.

## Getting Started

Clone the repository:

```bash
git clone https://github.com/Pranati-Shah/herspace-frontend.git
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```
